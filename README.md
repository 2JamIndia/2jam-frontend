## Стек технологий

-	TypeScript
-   React
-   Redux
-   node-sass
-   axios

## Установка

Клонируйте репозиторий.

```
git clone https://github.com/2jam/2jam-frontend.git
```

Переход в рабочий каталог.

```
cd 2jam-frontend
```

Установка зависимостей

```
npm install
```

Запуск сборки

```
npm start
```
